Metadata Schema Documentation

This document provides the detailed metadata schema developed for the DTL collections. The schema includes key elements to ensure accurate cataloging and retrieval of religious texts.

Metadata Elements

  • Title: The title of the resource.
  • Author(s): The authors of the resource.
  • Publication Date: The date when the resource was published.
  • Language: The language in which the resource is written.
  • Subject: The subject categories relevant to the resource.
  • Description: A brief description of the resource.
  • Publisher: The publisher of the resource.
  • Identifier: Unique identifiers such as ISBN or OCLC number.
  • Format: The format of the resource (e.g., PDF, Print).
  • Source: The source from which the resource is derived.
  • Rights: Information about the rights and permissions associated with the resource.
  • Collection ID: A unique identifier for the collection to which the resource belongs.

Example Metadata Record

{
    "title": "The Influence of Buddhism on Chinese Literature",
    "author": "Wang Wei",
    "publication_date": "2023-06-15",
    "language": "zh",
    "subject": ["Buddhism", "Chinese Literature"],
    "description": "This study explores the influence of Buddhism on Chinese literature during the Tang dynasty.",
    "publisher": "Peking University Press",
    "identifier": "ISBN 978-3-16-148410-0",
    "format": "PDF",
    "source": "Original manuscript housed at the National Library of China",
    "rights": "Creative Commons Attribution 4.0 International License",
    "collection_id": "123456"
}
