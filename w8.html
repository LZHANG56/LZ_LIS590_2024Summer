Week 8 Summary

Date: August 3 - 9, 2024

Activities

  • K-BART: Knowledge Base and Related Tools:
    • K-Bart Video: Watched the K-Bart video “Create WorldCat knowledge base collections” starting at 6:15. Link to video
  • Downloading a Collection:
    • Downloaded a collection as a tab-delimited .txt file.
    • Saved and opened the file in Excel for further work.
  • Uploading a Collection:
    • Added/edited titles as needed in the Excel file.
    • Changed the collection ID (column V) to the new collection.
    • Set Action (column Z) to autofill.
    • Saved the Excel sheet as a tab-delimited .txt file.
    • For Mac users or those using Excel on Microsoft 365 without the .txt option, copied content into a Google Sheet instead.
    • Uploaded the file via the “Collection Actions” dropdown by selecting “Upload Titles to This Collection.”
    • Selected “Scope” as needed and uploaded the file.
    • Viewed the status message and waited for the upload to process.
    • Viewed the preview by selecting the “View the report here” link.
    • Verified the number of titles added and clicked “Approve Upload.”
    • Waited for the collection to be indexed, after which the titles became visible in the collection.
  • Exercise:
    • Followed instructions to download one of the collections worked on and opened the downloaded .txt file as an Excel file.
    • Deleted all but the first 4-5 titles to speed up processing time.
    • Created a new collection with the Collection Name “[Your name] Training”.
    • Clicked “Open Access” and left other fields blank.
    • Retrieved the “Collection ID” from the “Properties” accordion to enter into Column V in the new spreadsheet.
    • Changed Column Z from “raw” to “autofill.”
    • Saved the file as a tab-delimited .txt file.
    • Uploaded the file following the aforementioned instructions.

Key Learnings

  • K-BART Tools: Gained practical experience with K-BART tools and the process of creating, downloading, and uploading collections.
  • Metadata Management: Improved skills in managing metadata, specifically in the context of WorldCat knowledge base collections.
  • Collection Processing: Understood the steps and necessary actions for processing and verifying collections.

Challenges

  • File Processing Time: Managing the processing time for larger files required patience and strategic deletion of unnecessary entries for the exercise.
  • Metadata Accuracy: Ensuring metadata accuracy and consistency across the collections was a meticulous and time-consuming task.

Next Steps

  • Refine Skills: Continue to refine skills in using K-BART tools and metadata management.
  • Expand Collections: Apply the learned techniques to expand and enhance other collections.
  • Seek Feedback: Engage with peers and mentors to gather feedback on the collection management process and improve further.

Examples
